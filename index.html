Bookmarker

Specifies the address to redirect to.
Specifies the URL of the icon. Can be a link to an ICO file or to a normal image.
Specifies the background color (in hex, without the hash symbol).
Empty means browser default.

Drop the bookmark here!

By dropping your existing bookmark here, you can edit it!

The usage is very simple. You input the URL you want to be redirected to, URL of the icon you want the bookmark to have, and optionally the background color for the fraction of a second it takes to redirect.

After clicking the Generate button, you get a URL to use for the bookmark instead of the original one!

PS. You can drag&drop the bookmark back here to import the settings!

This project is open-source on GitHub! Contributions are welcome!
